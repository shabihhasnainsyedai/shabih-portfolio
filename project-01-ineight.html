Overview

El-Seif Engineering & Contracting manages dozens of simultaneous live construction projects across Saudi Arabia. Prior to this initiative, project document management was fragmented — ACONEX handled document control while InEight was selected as the new project management platform, but no integration existed between the two. Thirty-five document controllers manually re-entered every document record between both systems, consuming 80–100 hours every week and introducing compounding data errors.

The mandate: implement InEight company-wide, build a live two-way REST API integration with ACONEX, and drive adoption across all active projects — without disrupting live construction operations.

Challenge & Opportunity
⚠ Challenge

35 document controllers were manually copying data between ACONEX and InEight — an error-prone, unscalable process consuming 80–100 hours per week. Platform adoption was also lagging, with project teams defaulting to spreadsheets instead of InEight.

✦ Opportunity

InEight's REST API capability provided the means to build a real-time two-way integration that would eliminate all manual entry. Combined with a structured adoption rollout, this could simultaneously fix the integration problem and drive company-wide platform maturity.

Solution Delivery
1

Requirements Mapping & Platform Configuration

Worked with project teams across all active sites to map workflows, configure InEight modules to El-Seif's construction processes, and define governance rules for document approvals, transmittals, and data standards.

2

Two-Way REST API Integration with ACONEX

Engineered a live bidirectional REST API integration between ACONEX and InEight's Document Module. Documents registered in either system now sync automatically in real time — zero manual intervention required from document controllers.

3

Phased Adoption Rollout Across Live Projects

Designed a project-by-project onboarding strategy that let live operations continue uninterrupted. Each project team was onboarded in sequence, with dedicated training and hypercare support through go-live.

4

Executive Dashboard Automation

Built automated Power BI reporting layers connected to InEight data, replacing weekly manual status reports with live project performance dashboards for leadership.

5

Digital Governance Framework

Architected a company-wide Digital Governance Framework anchored in HR policy — establishing clear data ownership, escalation paths, and platform accountability across all departments.
