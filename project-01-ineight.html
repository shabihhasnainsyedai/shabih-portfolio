Overview

El-Seif Engineering & Contracting manages dozens of simultaneous live construction projects across Saudi Arabia. Prior to this transformation, ACONEX handled document control while InEight was selected as the new project management platform — but no integration existed between the two. Thirty-five document controllers manually re-entered every document record between both systems, consuming 80–100 hours every single week and introducing compounding data errors across active project sites.

The mandate: implement InEight company-wide, engineer a live two-way REST API integration with ACONEX, and drive adoption across all active projects — without disrupting live construction operations for a single day.

Solution Delivery
1

Requirements Mapping & Configuration

Worked across all active project teams to map workflows, configure InEight modules to El-Seif's processes, and establish data governance rules for transmittals, approvals, and document standards.

2

Two-Way REST API with ACONEX

Engineered a live bidirectional REST API integration — documents registered in either system sync automatically in real time, zero manual intervention required.

3

Phased Project-by-Project Adoption

Designed a sequenced onboarding strategy so live operations continued without disruption. Each project team was onboarded with dedicated training and hypercare through go-live.

4

Executive Dashboard Automation

Built Power BI reporting layers connected to InEight data — replacing weekly manual status reports with live performance dashboards for leadership.

5

Digital Governance Framework

Architected a company-wide Digital Governance Framework anchored in HR policy — clear data ownership, escalation paths, and platform accountability across all departments.
