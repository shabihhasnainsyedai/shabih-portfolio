Overview

Metrolinx is Ontario's regional transportation authority, managing billions of dollars in transit infrastructure across the Greater Toronto and Hamilton Area. Their existing Oracle PCM platform had reached end-of-life, creating an urgent need to migrate to Oracle Unifier — without disrupting 120+ live projects or compromising data integrity on publicly-funded infrastructure.

As Migration Lead and Integration Manager, the mandate was to design and execute the full migration end-to-end: 6TB of contract data, project records, cost structures, and workflow configurations — all while live operations continued without interruption.

Challenge & Opportunity
⚠ Challenge

Migrating 120+ live projects meant there was no 'freeze the data' option — projects were actively running throughout. Data structures between Oracle PCM and Unifier were fundamentally different, requiring complex field-by-field mapping, transformation logic, and multi-layer validation. Any error could mean incorrect financial data on a live, publicly-funded transit program.

✦ Opportunity

Advanced SQL modelling could be used not just to execute the migration, but to analyse existing workflow patterns and surface hidden inefficiencies — turning a technical migration into a broader operational improvement that ultimately saved 250–300 hours and accelerated the program by 2–3 months.

Solution Delivery
1

Full Data Inventory & Structural Analysis

Performed a complete inventory of all 120+ Oracle PCM projects — mapping data structures, identifying field-level schema discrepancies, and categorising each project by complexity, data volume, and migration risk.

2

SQL Workflow Analysis & Efficiency Discovery

Built advanced SQL models to analyse existing data patterns across all projects. This uncovered critical workflow redundancies that, when resolved, saved 250–300 hours and accelerated the overall delivery timeline by 2–3 months.

3

ETL Pipeline Design & Transformation

Designed and built Extract-Transform-Load pipelines to migrate data from PCM to Unifier — including field mapping, data cleansing, schema transformation logic, and multi-layer validation to ensure 100% data integrity post-migration.

4

Phased Migration with Live Project Continuity

Executed migration in carefully sequenced phases — lower-risk projects first to validate the pipeline, then progressively moving higher-complexity projects while maintaining full operational continuity throughout.

5

Tableau & Power BI Rollout to 200+ Users

In parallel, led the rollout of Tableau and Power BI reporting to 200+ Metrolinx users — hands-on training driving full adoption and retiring legacy manual reporting processes.

6

P6 + ECOSYS Schedule-Cost Integration

Implemented automated integration between Primavera P6 and ECOSYS, eliminating 6–8 hours per week of manual schedule-cost reconciliation and delivering a unified real-time performance view.
