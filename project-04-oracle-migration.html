Oracle PCM → Unifier Migration (Metrolinx)

End-to-end migration of 120+ live projects and ~6TB of contract data from Oracle PCM to Oracle Unifier — delivered on time, within budget, and 2–3 months ahead of schedule through SQL-driven workflow analysis that surfaced 250–300 hours of hidden inefficiencies.

Overview

Metrolinx is Ontario's regional transportation authority, managing billions of dollars in transit infrastructure across the Greater Toronto and Hamilton Area. Their Oracle PCM platform had reached end-of-life — creating an urgent need to migrate to Oracle Unifier without disrupting 120+ live projects or compromising data integrity on publicly-funded infrastructure.

As Migration Lead, the mandate was clear: migrate ~6TB of contract data, project records, and cost structures across 120+ active programs — while live construction continued uninterrupted. What began as a migration became a broader operational improvement that saved 250–300 hours and accelerated the entire program by 2–3 months.

Challenge & Solution
⚠ Challenge

6TB of live data — no freeze window

120+ active projects running throughout the migration. Fundamentally different data structures between Oracle PCM and Unifier required complex field mapping, transformation logic, and multi-layer validation — zero tolerance for data errors on publicly-funded transit programs.

✦ Solution

SQL analysis + phased ETL migration

Advanced SQL modeling revealed hidden workflow redundancies that saved 250–300 hours and accelerated the program 2–3 months. Phased ETL migration ensured live operations continued uninterrupted with zero data loss.

2–3 Months

Ahead of the original program schedule

SQL workflow analysis turned a straightforward migration into a broader operational improvement — surfacing redundancies that unlocked 250–300 hours of savings and accelerated the full program timeline beyond what was planned.

Solution Delivery
1

Full Data Inventory & Structural Analysis

Complete inventory of all 120+ Oracle PCM projects — mapping schemas, identifying field discrepancies between PCM and Unifier, and categorising every project by complexity and migration risk.

2

SQL Workflow Analysis — 300 Hours Recovered

Built advanced SQL models to analyse existing data patterns. Uncovered critical workflow redundancies that saved 250–300 hours and accelerated the delivery timeline by 2–3 months.

3

ETL Pipeline Design & Transformation

Designed and built Extract-Transform-Load pipelines — field mapping, data cleansing, schema transformation, and multi-layer validation ensuring 100% data integrity.

4

Phased Migration with Live Continuity

Executed migration in sequenced phases — lower-risk projects first to validate the pipeline, then progressively migrating complex programs while maintaining full operational continuity.

5

Tableau & Power BI Rollout to 200+ Users

Led the enterprise rollout of Tableau and Power BI to 200+ Metrolinx users simultaneously — hands-on training driving full adoption and retiring legacy manual reporting.
